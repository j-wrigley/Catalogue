Catalogue is a lightweight, flat-file CMS built around clarity, speed, and flexibility. It uses simple files instead of databases and generates static pages that are easy to deploy anywhere, making it an approachable tool for designers and developers who want control without unnecessary complexity. Currently in beta, it continues to evolve as an open, adaptable system within the Idea Index ecosystem.

The documentation for Catalogue will be published across a set of structured Notion pages, offering clear sections for setup, blueprints, components, and deployment workflows.
