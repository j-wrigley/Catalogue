## Overview

The media management system provides:
- **Centralized storage** - All files in one location
- **Folder organization** - Organize files into folders
- **Metadata system** - Add alt text, captions, tags to files
- **Media picker** - Easy file selection in content forms
- **File operations** - Delete, move, copy URLs

## Media Library Location

Files are stored in:
```
/catalogue/uploads/
  images/
  documents/
  file.jpg
```

## Key Features

- **File uploads** - Single or multiple files
- **Folder management** - Create and organize folders
- **Metadata** - Alt text, captions, descriptions, tags
- **Media picker** - Visual file selection modal
- **Grid/List views** - Switch between view modes
- **File operations** - Delete, move, copy URL
