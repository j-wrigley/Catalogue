# Navigation & Sidebar

Understanding the admin panel navigation and sidebar.

## Overview

The sidebar provides quick access to all CMS sections. It can be collapsed to icon-only mode for more screen space.

## Sidebar Structure

### Main Navigation

- **Dashboard** - Overview and statistics
- **Pages** - Manage pages
- **Collections** - Manage collections
- **Media** - Media library
- **Site** - Site settings
- **Users** - User management

### Footer Navigation

- **View Site** - Open frontend site in new tab
- **Settings** - CMS settings
- **Logout** - Sign out

## Collapsing Sidebar

### Desktop

- Click **chevron button** at top of sidebar
- Sidebar collapses to icon-only mode
- Labels hidden, icons remain visible
- Click again to expand

### Mobile

- Sidebar always collapsed
- Icon-only mode
- Toggle button hidden
- More screen space for content

## Active State

- Current page highlighted
- Active link shows different color
- Icon and text both highlighted
- Clear visual indication

## Navigation Behavior

### Page Navigation

- Click navigation link
- Page loads immediately
- Active state updates
- Smooth transitions

### External Links

- **View Site** - Opens in new tab
- External link icon shown
- Opens frontend site

## Sidebar Features

### Icons

- Each section has unique icon
- Icons visible in collapsed mode
- Tooltips show on hover (collapsed)
- Consistent icon style

### Labels

- Text labels shown when expanded
- Hidden when collapsed
- Truncated if too long
- Responsive sizing

### Spacing

- Consistent spacing between items
- Footer separated from main nav
- Clean, organized layout

## Responsive Behavior

### Desktop (>768px)

- Sidebar always visible
- Can be collapsed/expanded
- Full labels shown when expanded

### Tablet (768px - 1024px)

- Sidebar can be collapsed
- More space for content
- Touch-friendly targets

### Mobile (<768px)

- Sidebar always collapsed
- Icon-only mode
- Toggle button hidden
- Optimized for small screens

## Keyboard Navigation

- **Tab** - Navigate through links
- **Enter** - Activate link
- **Escape** - Close modals/menus
- **Arrow keys** - Navigate lists

## Examples

### Collapsing Sidebar

1. Click chevron button (top left)
2. Sidebar collapses to icons
3. More space for content
4. Click again to expand

### Navigating Sections

1. Click "Pages" in sidebar
2. Pages section loads
3. "Pages" link highlighted
4. Content area updates

### Viewing Site

1. Click "View Site" in footer
2. Frontend opens in new tab
3. Can reference while editing
4. Easy to preview changes
