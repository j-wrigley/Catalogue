## Overview

The admin panel is the main interface for managing your CMS content. It provides a clean, modern interface for creating, editing, and organizing content.

## Accessing the Admin Panel

### Login

1. Navigate to `/catalogue/index.php?page=login`
2. Enter your username and password
3. Click **"Sign in"**

### First Access

If no users exist, you'll need to create your first user. See [User Management](../users/USER_MANAGEMENT.md) for details.

## Features

- **Clean Interface** - Minimal, modern design inspired by Radix UI
- **Responsive** - Works on desktop, tablet, and mobile
- **Collapsible Sidebar** - Icon-only mode for more space
- **Real-time Updates** - Changes reflect immediately
- **Toast Notifications** - Non-intrusive feedback
- **Keyboard Shortcuts** - Efficient workflow

## Quick Start

1. **Login** - Access the admin panel
2. **Dashboard** - View overview and statistics
3. **Create Content** - Add pages or collection items
4. **Manage Media** - Upload images and files
5. **Configure** - Set up site and CMS settings
