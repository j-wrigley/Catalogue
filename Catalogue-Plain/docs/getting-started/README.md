## Overview

JSON Catalogue CMS is a flat-file content management system that:
- **No database required** - All content stored as JSON files
- **Blueprint-driven** - Define content structure with YAML
- **Static generation** - Generates HTML files from templates
- **Modern admin panel** - Clean, intuitive interface
- **Secure** - Built-in security features

## Quick Start (5 Minutes)

1. **Install** - Upload files to your server
2. **Configure** - Set file permissions
3. **Login** - Access admin panel
4. **Create User** - Set up your first user account
5. **Create Content** - Add your first page

## Installation Types

### Root Installation

Install in your website's root directory:
```
/var/www/html/
  catalogue/
  assets/
  index.html
```

### Subfolder Installation

Install in a subfolder:
```
/var/www/html/
  mysite/
    catalogue/
    assets/
    index.html
```
