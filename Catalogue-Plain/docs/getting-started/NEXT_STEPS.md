## Common Tasks

### Creating a Blog

**Steps:**
1. Create `posts.blueprint.yml`
2. Create `templates/posts.php`
3. Create `templates/archive.php`
4. Add blog posts via Collections
5. Link archive page in navigation

**See:** [Collections Documentation](../templates/COLLECTIONS.md)

### Building Navigation

**Steps:**
1. Create navigation snippet
2. Use `catalogueNav()` function
3. Add to header template
4. Control page visibility

**See:** [Navigation Documentation](../templates/NAVIGATION.md)

### Adding Media

**Steps:**
1. Upload files to Media library
2. Organize into folders
3. Add metadata (alt text, captions)
4. Use in content via media picker

**See:** [Media Management](../media-management/README.md)

### Customizing Templates

**Steps:**
1. Edit template files
2. Use `catalogue()` functions
3. Add your own HTML/CSS
4. Create reusable snippets

**See:** [Templates Documentation](../templates/README.md)
